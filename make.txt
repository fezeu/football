notre project est baser sur angular, nodejs et mongodb.
etape pour lancer le systeme:
    -on lance mongodb
    -on importe la base de donnees:
    mongoimport -d footappbase -c collection_name collection_name.json
    -on installe les dependances du server nodejs puis on le lance:
     >cd server
     >npm install
     >node server.js
    -le vue est deja compilter
le code souce complet est disponible a l'address: https://github.com/fezeu/football
